# Datatypes

---

We will be exploring different datatypes

---

1. Integer
2. String
3. Float

---

## Integer

* Any whole number

## String

* Any letters, numbers, and characters
* Quotation marks are needed around the value

## Float

* Any number with a decimal

## Boolean

* True or False

``` python
type()
```
