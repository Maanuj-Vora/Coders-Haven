# Variables

---

Any item/name that is referencing/stores another item in storage

---

## Variables can have any name

For example

- x
- Maanuj
- Is_A
- Smart1e

---

## Instanitating a Variable

For example

> x = 5
>
> Maanuj = "Ok"
>
> Is_A = 0.01
>
> Smart1e = True

---

## Printing Variables

Place the variable name in the parenthesis of the print() function
